# VoiceKey

Sing in your perfect key. VoiceKey detects your vocal range in real-time and transposes any song to match your voice.

## Features
- Real-time pitch detection via Web Audio API
- Voice type classification (Bass → Soprano)
- Auto-transpose songs to your range
- Guitar chord diagrams
- Built-in song library

## Dev
```bash
npm install
npm run dev
```

## Deploy
Push to GitHub → connect to Netlify → auto-deploys from `main`.
